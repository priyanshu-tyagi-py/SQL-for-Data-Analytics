# SQL Interview Preparation

This repository contains a structured collection of SQL problems and solutions organized by topic. It is designed to support preparation for Data Analyst and related technical roles.

## Repository Structure

- 01_Joins
  Queries covering INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN, and self joins.

- 02_Aggregations
  Problems involving GROUP BY, HAVING, aggregate functions, and summary analysis.

- 03_Window_Functions
  Use of ROW_NUMBER, RANK, DENSE_RANK, LEAD, LAG, and partitioning techniques.

- 04_Subqueries
  Nested queries, correlated subqueries, and filtering using subqueries.

- 05_Case_Statements
  Conditional logic using CASE for transformation and categorization.

- 06_CTE
  Common Table Expressions for improving readability and handling complex queries.

## Objective

The objective of this repository is to:

- Build strong SQL fundamentals
- Practice interview-level problems
- Develop structured query-writing skills
- Improve problem-solving ability in data scenarios

## Approach

Each problem is documented with:

- Problem statement
- SQL solution
- Explanation of the logic used
- Alternative approaches where applicable

## Tools and Environment

- SQL (MySQL)
- Standard relational datasets for practice

## Usage

This repository can be used for:

- Interview preparation
- Revision of SQL concepts
- Practicing real-world data problems
